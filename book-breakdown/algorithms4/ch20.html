MSD 字符串排序
第 1 步：按第 1 个字符分成 R 个桶 第 2 步：每个桶内递归按第 2 个字符分 第 3 步：继续递归... 直到所有子数组有序 只需检查能区分字符串的那几个字符
基石：键索引计数法的四步魔法

键索引计数法是所有字符串排序的基础。它用 4 个简单步骤实现线性时间排序，完全不需要比较操作。

LSD vs MSD：两种截然相反的扫描方向
LSD（低位优先）
// 从最后一位到第一位 for (int d = W-1; d >= 0; d--) { // 对第 d 位做键索引计数 int[] count = new int[R+1]; for (int i = 0; i < N; i++) count[a[i].charAt(d) + 1]++; for (int r = 0; r < R; r++) count[r+1] += count[r]; for (int i = 0; i < N; i++) aux[count[a[i].charAt(d)]++] = a[i]; for (int i = 0; i < N; i++) a[i] = aux[i]; }
VS
MSD（高位优先）
// 从第一位开始，递归处理 private static void sort(String[] a, int lo, int hi, int d) { if (hi <= lo + M) { Insertion.sort(a, lo, hi, d); return; } int[] count = new int[R+2]; for (int i = lo; i <= hi; i++) count[charAt(a[i], d) + 2]++; // ... 键索引计数 ... for (int r = 0; r < R; r++) sort(a, lo+count[r], lo+count[r+1]-1, d+1); }
LSD 要求所有字符串等长，做 W 轮键索引计数，依赖稳定性保证正确性。MSD 适用于变长字符串，通常只需检查区分字符串的几个字符即可，但需要处理大量空桶和「字符串结束」的特殊情况。
