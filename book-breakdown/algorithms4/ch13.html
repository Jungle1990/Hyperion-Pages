左旋 vs 右旋：红黑树的两把扳手
左旋 rotateLeft(h)
Node rotateLeft(Node h) { Node x = h.right; h.right = x.left; x.left = h; x.color = h.color; h.color = RED; x.N = h.N; h.N = 1 + size(h.left) + size(h.right); return x; } // 场景: 右链接是红色 -> 转为左倾 // h 下沉, x 上升, 红色转移到左边
VS
右旋 rotateRight(h)
Node rotateRight(Node h) { Node x = h.left; h.left = x.right; x.right = h; x.color = h.color; h.color = RED; x.N = h.N; h.N = 1 + size(h.left) + size(h.right); return x; } // 场景: 连续两个左红链接 -> 右旋上层 // h 下沉, x 上升, 修正连续红链接
两个操作完全对称，只是 left/right 互换。关键性质：旋转保持有序性和完美黑色平衡。返回的链接继承原来 h 的颜色（x.color = h.color），被旋转下去的节点变成红色（h.color = RED）。
