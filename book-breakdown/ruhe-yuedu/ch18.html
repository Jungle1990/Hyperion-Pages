哲学的两大领域
理论 / 思辨哲学
核心问题：世界是什么样的？ • 形上学 → 存在的特质 • 自然哲学 → 变化的本质与原因 • 认识论 → 人类知识的起因与限制 关键问题举例： 存在与不存在的区别是什么？ 所有事物都会改变吗？ 有什么东西是永恒不变的？
VS
规范 / 实用哲学
核心问题：我们该做什么？ • 伦理学 → 如何过好生活 • 政治哲学 → 好社会与正义 关键问题举例： 善与恶的区别是普遍适用的吗？ 善行之间有等级之分吗？ 对的事一定是善的吗？ 错的事一定是恶的吗？
第一顺位问题 = 关于世界与人生的根本问题 第二顺位问题 = 关于我们如何思考和表达的问题 现代哲学大多只关注第二顺位问题，这让门外汉更难阅读了。
哲学论述的五种风格

从柏拉图到尼采，哲学家们发展出五种截然不同的写作方式。了解这些风格能帮你选择正确的阅读策略。

🧠
哲学家面对问题时，除了思考以外什么也不能做。读者面对哲学书时，除了阅读以外什么也不能做——那也就是说，要运用你的思考。 这种「必要的孤独」——存在于读者与一本书之间——正是分析阅读的规则用在哲学书上比其他书更适用的原因。 哲学诉求的是你的普通常识与日常观察，不需要特殊实验或专业经验。你用的方法和哲学家写作时用的方法是一样的。
