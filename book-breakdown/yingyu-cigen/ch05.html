🔤
词根的构词
零级派生 → 词根不加任何词缀 一级派生 → 粘附词根 + 原生词缀(进口配饰) 二级派生 → 自由词根 + 派生词缀(国产配饰) 不管怎么组合，词根始终是核心
复杂的词不过是多穿了几件衣服：predilection = pre-(事先) + di-(出) + lect(选) + -ion(行为) = 「事先就选出了偏好」= 偏爱。层层剥开，核心词根 lect(选) 就在那里。
三级派生法一览
零级派生的两种情况
VS
传统构词法中的「转化」和「合成」其实都是零级派生——词根不加任何词缀的构词方式。只是「转化」用一个自由词根，「合成」用两个。
一级派生：粘附词根 + 原生词缀的四种组合
传统构词法 vs 词素构词法
VS
传统构词法只看到「分子」层面(单词)，词素构词法深入到「原子」层面(词素)。前者能解释 blackboard = black + board，后者还能解释 television = tele(远) + vis(看) + -ion(状态)。
