vert/vers: 一个「转」字的百变面孔
🔄
vert/vers = to turn（转）
不同前缀 = 不同方向的「转」 产生了极其丰富的词汇家族
⟺
🌀
转的各种方向
convert = con(完全)转 → 转变 reverse = re(回)转 → 反转 advertise = ad(向)转 → 让人转过来看 → 广告 diverse = di(分开)转 → 转向不同方向 → 多样的 controvert = contro(反对)转 → 反着转 → 争论 pervert = per(歪)转 → 转歪了 → 曲解
advertise（广告）的字面意思是「让人转过来看」——ad(向)+vert(转)→引起注意。universe（宇宙）=uni(一)+vers(转)→统一的旋转体→宇宙。
