spec/spect: 一个「看」字的万花筒
👁️
spec/spect = to look（看）
加不同前缀 = 从不同角度「看」 每个方向都对应一个常用词
⟺
🔄
七个方向的「看」
expect = ex(向外)看 → 期待 inspect = in(向内)看 → 检查 respect = re(再次)看 → 尊重 suspect = su(从下)看 → 怀疑 prospect = pro(向前)看 → 前景 retrospect = retro(向后)看 → 回顾 perspective = per(穿过)看 → 透视
expect(期待)是「向外张望」——等待某件事出现。respect(尊重)是「再看一眼」——值得再看的人才值得尊重。suspect(怀疑)是「从下偷看」——鬼鬼祟祟地窥探。
词根拆解：inspire（灵感）的神秘来源

💡 sid = to sit（坐），pre + sid + ent = 坐在前面的人
