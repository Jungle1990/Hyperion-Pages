核心类比：两种插座
🔌
派生词缀(国标插座)
能与自由词根/单词结合 如 un- + friendly → unfriendly 如 work + -er → worker 本族语和已同化的外来语词缀 读者通常已经掌握
⟺
🔌
原生词缀(进口插座)
只能与粘附词根结合 如 pro- + vid(看) + -e → provide 如 vis(看) + -ible → visible 来自拉丁/希腊的古典语词缀 是词缀学习的重点和新大陆
家里70%的电器(词汇)是进口的(古典语源)，如果只会用国标插座(派生词缀)，那70%的电器都用不了。学会进口插座(原生词缀)，才能让所有电器正常工作。
