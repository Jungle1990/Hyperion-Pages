ject的投掷方向：前缀决定一切
🎯
ject = to throw（投掷）
一个核心动作 不同前缀 = 不同方向 = 完全不同的词义
⟺
🔄
前缀 + 投掷 = 新词
inject = in(向内)扔 → 注射 reject = re(回)扔 → 拒绝 project = pro(向前)扔 → 投射/项目 subject = sub(下面)扔 → 使服从/主题 object = ob(对着)扔 → 反对/物体
一个「扔」字，五个前缀，五种完全不同的含义。这就是词根构词法的魅力——核心动作不变，方向改变一切。
